Key Improvements:

  • Three-line structure is enforced with block spans
  • Icons use em units (1.2em height) for proportional scaling
  • Icon container has fixed width of 5.5ch to prevent layout shifts
  • No fixed pixel sizes - everything scales with font size
  • Vertical alignment uses vertical-align: middle instead of positioning
  • Clean, predictable layout at all screen sizes
