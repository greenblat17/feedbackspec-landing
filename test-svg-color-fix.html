How the Fix Works

The Problem:

SVGs use `fill="currentColor"` which inherits the text color. On dark backgrounds, black icons are invisible.

The Solution:

Applied CSS filter: `brightness(0) invert(1)` which makes the icons white on dark backgrounds.

Implementation:

style={{ 
  height: 'auto', 
  filter: 'brightness(0) invert(1)' 
}}
